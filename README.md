# My_Web
Spring 을 통한 웹 개발

백엔드 구현 목표 
1. MySQL 연동하여 회원가입 기능 - 기능 구현 완료
2. 회원가입 후 로그인 기능 - 미구현
3. 로그인 후 회원 정보 확인 및 수정 기능 - 미구현
4. 로그인 후 메일 발송 및 수신 기능 - 미구현
5. 자유게시판 및 Q/A 게시판

프론트 구현 목표
1. 기존 페이지 삭제 후 개인 프로필 넣기
2. 회원 대상 실시간 채팅 할 수 있도록 물론 이건 백엔드 연동 예정
